# AI Resume Optimizer

AI-powered Resume Optimizer developed as part of a technical assignment under Indian Navy WESEE.

## Features

* Upload Resume (PDF/DOCX)
* Extract text from resume
* Input Job Description
* Ready for LLM-based optimization

## Tech Stack

* Python
* Streamlit
* pdfplumber
* python-docx

## How to Run

```bash
pip install -r requirements.txt
streamlit run app.py
```
